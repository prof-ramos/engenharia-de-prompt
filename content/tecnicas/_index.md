# Técnicas de Prompting

Aprenda as principais técnicas para criar prompts eficazes.

## Técnicas Básicas

| Técnica | Descrição |
|---------|-----------|
| [Zero-shot](/tecnicas/zero-shot/) | Prompt simples, sem exemplos |
| [Few-shot](/tecnicas/few-shot/) | Com exemplos para guiar o modelo |

| Técnica | Descrição |
|---------|-----------|

| [Prompt de Papel](/tecnicas/prompt-papel/) | Atribui identidade/especialidade |
| [Prompt Contextual](/tecnicas/prompt-contextual/) | Fornece informações específicas |

## Técnicas de Raciocínio

| Técnica | Descrição |
|---------|-----------|
| [Step-back](/tecnicas/step-back/) | Recuo para abstração |
| [Cadeia de Pensamento](/tecnicas/cot/) | Raciocínio explícito passo a passo |
| [Auto-consistência](/tecnicas/auto-consistencia/) | Votação majoritária |
| [Árvore de Pensamentos](/tecnicas/tot/) | Exploração de múltiplos caminhos |
