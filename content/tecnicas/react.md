# ReAct (Raciocinar e Agir)

Combina **raciocínio** com **uso de ferramentas** externas.

## Ciclo ReAct

```
1. Thought (Pensamento): O que preciso fazer?
2. Action (Ação): Qual ferramenta usar?
3. Observation (Observação): O que aconteceu?
4. Repete até completar
```

## Ferramentas Comuns

| Ferramenta | Uso |
|------------|-----|
| Search | Buscar informações na web |
| Calculator | Cálculos matemáticos |
| Code | Executar código |
| API | Chamar APIs externas |

## Benefícios

- Informações atualizadas em tempo real
- Cálculos precisos
- Rastreabilidade
- Flexibilidade

## Quando Usar

- Informações que mudam
- Cálculos complexos
- Integração com APIs
- Agentes autônomos

## Dicas

1. Limite o número de passos
2. Defina ferramentas claramente
3. Trate erros das ferramentas
4. Monitore custos
