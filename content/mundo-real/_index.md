# Prompts no Mundo Real

Esta seção é baseada no curso **"Real World Prompting"** da Anthropic, adaptado e traduzido para português.

## O que você vai aprender

Aplicar técnicas de prompting em **casos reais de produção**, incluindo:

- Como estruturar prompts complexos
- Processo de iteração e refinamento
- Exemplos práticos completos (médico, resumo de chamadas, suporte)
- Framework de tomada de decisão

## Diferença entre "Prompting Básico" e "Engenharia de Prompt"

| Básico | Engenharia |
|--------|------------|
| Interação única | Multi-turno, contexto persistente |
| Consultas simples | Instruções complexas |
| Vago ou ambíguo | Preciso, sem espaço para erro |
| One-off | Iterativo, sistemático |
| Casual | Produção, escalável |
