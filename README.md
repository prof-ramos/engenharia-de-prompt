# Engenharia de Prompt

Documentação completa sobre engenharia de prompt para LLMs, baseada no whitepaper do Google/Gemini.

## Conteúdo

- **Configurações de LLM**: Temperatura, Top-K, Top-P
- **Técnicas de Prompting**:
  - Zero-shot e Few-shot
  - Prompts de Sistema, Contextuais e de Papel
  - Step-back (Recuo)
  - Cadeia de Pensamento (CoT)
  - Auto-consistência
  - Árvore de Pensamentos (ToT)
  - ReAct (Raciocinar e Agir)
  - Engenharia Automática de Prompts (APE)
- **Prompting para Código**: Escrever, explicar, traduzir e depurar
- **Melhores Práticas**: 15 práticas essenciais
